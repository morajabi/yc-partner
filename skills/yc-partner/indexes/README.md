# Indexes

Indexes map user tasks to the smallest relevant guides and source notes.

Use indexes as routing tables, not source evidence. For source-grounded claims, follow the index link to the underlying guide or processed source note.

- `application-questions.md` - route by YC application prompt or form area.
- `faq.md` - route common pre-application doubts, logistics, rejection, and reapplication questions.
- `interview.md` - route YC interview prep and pressure-test tasks.
- `response-assets.md` - route founder-facing output assets: tips, quotes, facts, examples, sample rewrites, questions, and office-hours notes.
- `source-map.md` - high-level folder and source navigation.
- `themes.md` - route cross-cutting review issues such as clarity, users, traction, founder signal, competitors, office-hours obstacles, fundraising, and AI.
