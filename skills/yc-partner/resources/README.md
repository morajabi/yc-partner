# Resources

This folder contains processed Markdown source files only.

Every source file should include:

- Title.
- Source type.
- Source status.
- Original public URL.
- Author/channel, when known.
- Published date, when known.
- Processed date.
- Full processed Markdown text.
- Optional short summary.
- Extracted guidance or relevance to YC applications/interviews, when useful.

Do not commit raw HTML, downloaded media, raw transcripts, caption JSON, screenshots, private applications, or personal founder data.
