# YC Requests for Startups

Processed Markdown captures of official YC Requests for Startups sections from `https://www.ycombinator.com/rfs`.

Files are named as:

```text
<batch>-<section-slug>.md
```

Example:

```text
summer-2026-ai-for-low-pesticide-agriculture.md
```

Each file includes the original section anchor URL, author metadata when available, the batch label, and the full processed RFS text. Use these files as official YC context for markets and startup areas YC has publicly requested, while preserving YC's caveat that RFS ideas are only a fraction of what YC funds.
