# Tweets

Public tweets and X threads from YC-affiliated people and other relevant startup sources.

Use this folder for short-form public social posts that are useful as YC founder/partner or startup-source context. Each file should preserve the pasted or captured public text and include the original public URL when available.

Do not downgrade pasted public post text as secondhand, excerpted, or independently unverified. If the capture path matters, record it neutrally in compact metadata, for example `Method: pasted public text`.
