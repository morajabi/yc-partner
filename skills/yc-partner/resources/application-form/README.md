# YC Application Form Questions

Processed Markdown captures of official YC application form questions.

Use this folder for public, official YC application prompts and form guidance, especially when the exact wording matters for application review.

## What Belongs Here

- Official YC application questions copied from public YC application pages, a public official application form, or sanitized official YC application UI text.
- Batch-specific captures, such as `summer-2026-application-questions.md`, when the wording is tied to a known batch.
- Notes about changed, removed, or newly added official questions when captured from a public source.

## What Does Not Belong Here

- Private founder answers or drafts.
- Screenshots, raw HTML, browser session captures, cookies, or authenticated-only output beyond the sanitized question text itself.
- Founder answers, profile values, account names, emails, phone numbers, uploaded files, URLs from a private profile, validation state, or login/session details.
- Unofficial advice about how to answer the questions. Put that in guides or references instead.
- Public example applications. Put those in `../examples/`.

## Metadata

Each source file should use compact metadata:

- Type: `Official YC application form questions`.
- Status: `Official YC source`.
- URL.
- Batch or application cycle, when known.
- Captured date.
- Method, when useful.

Use the single `Summary` section for concise privacy/capture notes. Put answer guidance in indexes or guides.

## Naming

Prefer batch-specific filenames when possible:

```text
summer-2026-application-questions.md
fall-2026-application-questions.md
```

If the batch is unknown, use:

```text
current-application-questions-YYYY-MM-DD.md
```
