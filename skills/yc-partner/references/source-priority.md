# Source Priority

Use this order when grounding advice:

1. Official YC pages, guides, blog posts, Startup School material, and official YC YouTube videos.
2. Public essays, blog posts, talks, and interviews by YC partners, Paul Graham, Jessica Livingston, and similar YC-affiliated sources.
3. Public example YC applications and interviews.
4. Founder stories and anecdotes from successful YC founders, used as illustrative context only.
5. The agent's own inference from the user's application.

Label the source class when it matters. Do not present founder anecdotes as YC evaluation criteria.

If no source in the repository supports a YC-specific claim, say that the recommendation is an inference.
